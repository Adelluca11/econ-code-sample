# Applied Economics Coding Samples

This repository contains selected coding samples illustrating applied econometric and computational methods relevant for research in economics.

---

## 1. Difference-in-Differences with Heterogeneous Treatment Effects (R)

Folder: `did_heterogeneity/`

This project demonstrates:

- Fixed-effects difference-in-differences estimation  
- Binary and continuous heterogeneous treatment effects  
- Hypothesis testing of subgroup differences  
- Marginal effect visualization using binning estimators  

The implementation reflects a standard applied microeconometric workflow using panel data.

---

## 2. Reinforcement Learning – Q-Learning Implementation (Python)

Folder: `q_learning/`

This notebook implements:

- A Q-learning algorithm  
- Simulation-based learning dynamics  
- Convergence of value functions  
- Policy improvement through iterative updates
